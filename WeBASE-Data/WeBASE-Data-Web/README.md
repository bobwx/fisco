# 数据监管平台

## 简介
WeBASE-Data-Fetcher为数据监管平台，主要提供以下功能：

- 链上的交易数量和交易详情读取。
- 关键字搜索读取，查询交易属于哪条链，哪个机构，哪个合约。保证链上数据可查可管。

详细介绍请查看[WeBASE-Data在线文档](https://webasedoc.readthedocs.io/zh_CN/latest/docs/WeBASE-Data/index.html)
