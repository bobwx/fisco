## 简介
WeBASE-Transaction是交易上链代理子系统。主要接收无状态交易请求，缓存到数据库中，再异步上链。本系统可大幅提升吞吐量，解决区块链的tps瓶颈。 详细介绍请查看[WeBASE-Transaction在线文档](https://webasedoc.readthedocs.io/zh_CN/latest/docs/WeBASE-Transaction/index.html)
